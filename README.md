# AQI Prediction System - Delhi

## Overview
This project is a Machine Learning based AQI (Air Quality Index) Prediction System built using XGBoost and deployed with Streamlit.

The system:
- Predicts AQI using pollutant parameters
- Displays interactive visualizations
- Shows model performance (R² Score)
- Provides 7-day AQI forecast
- Includes KPI cards and AQI gauge meter

---

## Tech Stack
- Python
- XGBoost
- Scikit-learn
- Plotly
- Streamlit

---

##  Features
✔ Interactive Actual vs Predicted Graph  
✔ AQI Time-Series Visualization  
✔ KPI Dashboard  
✔ AQI Gauge Meter  
✔ Sidebar Prediction Panel  
✔ 7-Day Forecast
